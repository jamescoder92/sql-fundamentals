---------------------------------------------------------------
-- Why Filter Data?
---------------------------------------------------------------

-- Imagine your employees table contains
-- hundreds of records.
--
-- Usually you don't need all of them.
--
-- Instead, you ask SQL to return only
-- the rows that satisfy a condition.

---------------------------------------------------------------
-- WHERE
---------------------------------------------------------------

-- SQL filters rows using:
--
-- WHERE
--
-- General syntax:
--
-- SELECT ...
-- FROM ...
-- WHERE condition;

---------------------------------------------------------------
-- Comparison Operators
---------------------------------------------------------------

-- =
-- Equal to
--
-- !=
-- Not equal
--
-- >
-- Greater than
--
-- <
-- Less than
--
-- >=
--
-- <=

---------------------------------------------------------------
-- Logical Operators
---------------------------------------------------------------

-- AND
--
-- Every condition must be true.
--
---------------------------------------------------------------

-- OR
--
-- At least one condition must be true.
--
---------------------------------------------------------------

-- NOT
--
-- Negates a condition.

---------------------------------------------------------------
-- BETWEEN
---------------------------------------------------------------

-- Useful for ranges.
--
-- General syntax:
--
-- BETWEEN value1 AND value2

---------------------------------------------------------------
-- IN
---------------------------------------------------------------

-- Useful when matching one value
-- from a list.
--
-- General syntax:
--
-- IN (...)


---------------------------------------------------------------
-- LIKE
---------------------------------------------------------------

-- Used for pattern matching.
--
-- %
--
-- Zero or more characters.
--
---------------------------------------------------------------
--
-- _
--
-- Exactly one character.

---------------------------------------------------------------
-- NULL
---------------------------------------------------------------

-- NULL represents missing information.
--
-- SQL checks NULL using:
--
-- IS NULL
--
-- and
--
-- IS NOT NULL
